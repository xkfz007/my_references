76 /**

78  * store several packets (as interpreted by the codec) in a single container
79  * packet, which means the demuxer holds some back when the first container

83  *
84  * @param s context containing RMContext and AVIOContext for stream reading

89  * @return the number of samples left for subsequent calls to this same
90  * function, or 0 if all samples have been retrieved.
